E-Svd

Native application which enables the preview of everyday newspapers. App gives an option to review archived issues.

Design Problems

  • Difficult navigation.
  • Different preview modes and ways to present one newspaper issue.
  • Application for older users requires intuitive way of navigation.
  • Mess in hierarchy, sophisticated access path to articles.

Goals

  • Application easy to use for older users, basic functions visible on the main screen.
  • Easy switch between issues, swiping between articles.
  • Contrast between text and background to improve readability.
  • Simplified structure of application to eliminate unnecessary steps.
