Quiz

Web application for displaying quizzes. Project contains also user-firendly admin panel for editors. My responsibility was to plan MVP version of quiz app, improve and redisign admin panel and plan new features and improvments.

Design Problem

  • Old application hard to use , lot of unnecessery clicking between questions.
  • Difficult to use on mobile, small radio buttons to choose correct answer.
  • There is no connection between other parts of news service, no user flow.
  • Editors complain a lot for difficult admin panel its seems to be difficult to creat new quiz.

Goals

  • Reduce number of clicks
  • Design connection between other quizzes and services.
  • Focus on easy quiz sharing across social media.
  • Create new user-friendly admin panel which will improve adding new quizzes
