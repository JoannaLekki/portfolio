Quiz

Web application for displaying quizzes. The project also contains user-friendly admin panel for editors. My responsibility was to plan MVP version of quiz application, improve and redesign admin panel and plan new features and improvements.

Design Problems

  • Lot of unnecessary clicking between questions.
  • Difficult to use on mobile, small radio buttons for choosing correct answer.
  • No connection between other parts of the news service.
  • Too complicated admin panel, new quiz difficult to create.

Goals

  • Reduce number of clicks.
  • Build connection between other quizzes and services.
  • Focus on easy quiz sharing across social media.
  • Create new user-friendly admin panel which will improve adding new quizzes.

Final Solution

  • Application reduces number of clicks, first question shows just after clicking teaser.
  • Answers displayed as big buttons, easy to find and move to the next question.
  • Related quizzes promoted under the question and on the result page.
  • Links to the other service parts build better user flow.
  • Admin panel as a Single Page Application with noticeable list of questions, simple toggles to select wrong and correct answer.
